**ShoppyGlobe E-Commerce Application**

# Project Overview
ShoppyGlobe is a basic e-commerce application built with React and Redux. It allows users to:
1.Browse a list of products fetched from an API.
2.View detailed information about a product.
3.Add products to the cart, update quantities, or remove items.
4.Proceed to checkout using a dummy form.
5.Place an order and reset the cart.
6.Search and filter products dynamically.

# Features
- Product list with API data fetching (https://dummyjson.com/products)
- Product detail page with dynamic routing
- Shopping cart with quantity management and removal
- Checkout page with a dummy form and order placement
- 404 NotFound page for invalid routes

# Technology Features
1.React v18 with Vite
2.Redux Toolkit for state management
3.React Router DOM with dynamic routes
4.Custom hook (useFetchProducts) for API calls
5.Code splitting and lazy loading of components
6.Responsive CSS styling
7.Proper error handling for API requests
8.Full comments and proper indentation for code readability

# # Installation and Run
1.clone the repository:
git clone https://github.com/KhushbuKumari21/ShoppyGlobe-E-commerce-Application-.git
cd ShoppyGlobe

2.Install dependencies:
npm install

3.Start the development server:
npm run dev

url of Project:   http://localhost:5173/

# Usage Instructions
1.Home Page : Browse products, search with the input bar.
2.Product Detail :Click on a product to see details and add to cart.
3.Cart : Adjust quantities, remove items, or proceed to checkout.
4.CheckOut : Fill dummy user details and place order. Cart will reset and redirect to Home.
5.404 Page: Displayed if an invalid route is accessed.
